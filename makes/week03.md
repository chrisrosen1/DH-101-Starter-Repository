## Reflection
For this project, I created an AI-generated selfie using DALL·E in ChatGPT. I did not upload a real photo of myself. Instead, I described myself using short and general text prompts. The first prompt was: “Show a 24-year-old white man with defined facial features and hazel eyes.” Because the description was basic, the image did not really look like me. It looked like a person with ideal features and perfect proportions.
The face matched the traits I listed, but it felt generic. The image looked too perfect. There were no imperfections such as uneven features, a larger nose, smaller eyes, or different-sized ears. These details help make a real person look unique. Without them, the image looked perfect and could represent almost anyone.
The second prompt did not add more personal details. Instead, it created a more fun and cartoon-like version of the same person. The prompt placed him in a bright grass field with sunshines, rainbows, and candy. The result looked like a character from an animated movie. Even though the style changed, the person still looked perfect and idealized.
This project helped me understand that AI often creates clean and flawless versions of people. It does not naturally include the small differences that make someone real. Authenticity comes from recognizing what is missing and what makes each person special. Every person is unique and has imperfections that make their own identity.
